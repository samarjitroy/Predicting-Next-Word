The Objective

The main goal of this capstone project is to build a shiny application that is able to predict the next word while typing. Analysis and product was based on The swiftkey data.

This exercise was divided into three sub tasks as follows:

  1. Data cleansing, exploratory analysis, the creation of a predictive model and test with the test data.Click to view
  2. Creating n-gram tockenized data for Shiny Apps use. Click to view
  3. Creating a Shiny Apps Data Product Run Shiny App
  4. Creating Slide Deck to present the exercise

The Applied Methods & Models

While creating a sample(Training/Test) from the swiftkey source, the data was cleaned by conversion to lowercase, removing punctuation, links, white space, numbers and all kinds of special characters. The sample was then tokenized into n-grams.

The aggregated bi-gram,tri-gram and quad-gram term frequency matrices have been transferred into data frame.

The resulting data.frames are used to predict the next word in connection with the text input by a user of the shiny apps.

The Usage Of The Application

The user interface of Shiny App has user input box to type and print suggested word. While entering the text (1), the field with the predicted next word (2) refreshes instantaneously.
